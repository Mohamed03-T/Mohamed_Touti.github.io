اسحب للأسفل

👨‍💻 نبذة عني

محمد توتي مبرمج ومصمم مواقع إلكترونية محترف، متخصص في تطوير تطبيقات الويب وسطح المكتب والهاتف. أؤمن بأن التفاصيل هي التي تصنع الفرق، وأسعى دائمًا لدمج الإبداع مع الأداء لتحقيق تجارب مستخدم مميزة. لدي خبرة واسعة في بناء تطبيقات متكاملة باستخدام React, Next.js, JavaFX, Java EE, Prisma, MySQL, TypeScript، مع اهتمام خاص بتحقيق تجربة مستخدم سلسة وتصميم واجهات احترافية. كمؤسس لفريق Medev Team، أدير فرق تطوير متكاملة وأعمل عن قرب مع العملاء لتحقيق حلول مبتكرة تلبي احتياجاتهم. من أبرز مشاريعي: نظام تتبع الحافلات: تطبيق متكامل يعتمد على JavaFX وMySQL مع API مخصص، يقدم حلولًا دقيقة لتتبع المركبات وإدارة الرحلات بكفاءة. متجر إلكتروني متكامل: تم تطويره باستخدام Next.js وPrisma، مع دعم لأحدث تقنيات المصادقة وتكامل قواعد البيانات لضمان تجربة تسوق مميزة وسريعة. تطبيق تجارة الجملة: تطبيق مشابه لمنصة السمار، يتيح لأصحاب الأعمال بيع المنتجات بالجملة بسهولة ومرونة. أطمح لتطوير مهاراتي في الأتمتة بالذكاء الاصطناعي وتقديم حلول تقنية متقدمة تساهم في تغيير صناعة البرمجيات.

💼 الخبرة العملية

🔹 مبرمج ومطور تطبيقات (Software Developer)

  • خبرة في تطوير تطبيقات سطح المكتب باستخدام JavaFX مع تصميم واجهات احترافية وتجربة مستخدم مميزة.
  • تطوير تطبيقات الويب باستخدام Java EE مع دعم للتكامل مع قواعد البيانات MySQL.
  • بناء تطبيقات Android (قيد التعلم) مع التركيز على الأداء وتجربة المستخدم.
  • تصميم وتطوير أنظمة إدارة بيانات متكاملة باستخدام MySQL + API لتحسين سرعة الوصول إلى البيانات ومعالجتها.
  • تطوير تطبيقات مبتكرة مثل نظام تتبع الحافلات لتحليل حالات الحافلات وتتبع عدد الركاب بكفاءة.

🔹 التجارة والتسويق الإلكتروني (E-commerce & Digital Marketing)

  • خبرة في التسويق الإلكتروني والتسويق بالعمولة باستخدام أدوات متقدمة.
  • تحليل البيانات لتحسين حملات التسويق وزيادة العائد على الاستثمار.
  • استخدام أدوات الذكاء الاصطناعي لأتمتة العمليات التسويقية وتحليل سلوك العملاء.
  • تصميم وإطلاق متاجر إلكترونية باستخدام Next.js, Prisma, Tailwind CSS مع دعم كامل لتجربة المستخدم.

🔧 المهارات التقنية

💻 تطوير التطبيقات

  • JavaFX: بناء تطبيقات سطح المكتب بواجهات احترافية وتجربة مستخدم مميزة.
  • Java EE: تطوير تطبيقات الويب باستخدام Java مع دعم للتكامل مع قواعد البيانات.
  • Android (قيد التعلم): تصميم وتطوير تطبيقات الهاتف الذكي باستخدام Android Studio مع التركيز على الأداء وتجربة المستخدم.

🗄️ إدارة قواعد البيانات

  • MySQL: تصميم وإدارة قواعد بيانات قوية، مع التكامل مع API لإنشاء أنظمة متكاملة وموثوقة.
  • Prisma: استخدام Prisma لربط قواعد البيانات بتطبيقات Next.js وتحقيق سرعة وكفاءة عالية في معالجة البيانات.

🎨 التصميم وتجربة المستخدم

  • تصميم واجهات احترافية بألوان هادئة تعبر عن هوية العلامة التجارية.
  • تحقيق تجربة مستخدم سلسة من خلال تحسين تدفق العمليات وتقليل التعقيد.
  • استخدام Tailwind CSS وShadcn/ui لإنشاء تصاميم حديثة ومتجاوبة.

🤖 الأتمتة والذكاء الاصطناعي

  • أتمتة العمليات باستخدام أدوات الذكاء الاصطناعي لتحسين الكفاءة وتقليل التكاليف.
  • تحليل البيانات باستخدام الذكاء الاصطناعي لتحسين تجربة المستخدم واتخاذ قرارات ذكية.
  • بناء أنظمة تتبع وتحليل تعتمد على الذكاء الاصطناعي لتحسين أداء التطبيقات.

🚀 المشاريع الحالية

🚌 تطوير تطبيق تتبع الحافلات (Bus Tracking Application)

  • نظام متكامل لإدارة الحافلات وتتبع عدد الركاب بدقة.
  • حساب الإيرادات اليومية وتحليل أداء الحافلات لتحسين الكفاءة.
  • يعتمد على MySQL + API مع هيكلة مخصصة لضمان سرعة الوصول للبيانات واستقرارية الأداء.
  • دعم مستقبلي للتكامل مع تطبيقات الهواتف لتمكين أصحاب الحافلات من مراقبة الأساطيل بسهولة.

🤖 استكشاف مجال الأتمتة بالذكاء الاصطناعي (AI Automation)

  • البحث والتعلم عن أحدث أدوات الأتمتة لتحسين العمليات وتقليل التكاليف.
  • استخدام الذكاء الاصطناعي لتحليل البيانات والتنبؤ بالاتجاهات.
  • تجربة تقنيات مثل RPA (Robotic Process Automation) وAI Chatbots لتحسين تجربة العملاء.
  • تطوير حلول مبتكرة لتحليل البيانات وتحسين أداء التطبيقات.
